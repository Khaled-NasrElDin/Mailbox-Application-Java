# Mailbox Application

Java email client application with core mailbox features and a graphical user interface.

## Overview

This project is a Java-based mailbox application developed to simulate the main features of an email client. It includes common email actions such as composing, sending, forwarding, replying, replying to all, and deleting messages.

The project also includes a graphical user interface built with JavaFX.

## Features

- Compose emails
- Send emails
- Forward emails
- Reply to emails
- Reply all
- Delete emails
- Graphical user interface

## Technologies

- Java
- JavaFX

## Skills Demonstrated

- Object-oriented programming
- GUI development
- Event handling
- Application logic
- User interaction design
